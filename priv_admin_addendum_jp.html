アプリケーションプライバシーポリシー補足条項

第1条（定義）

このプライバシーポリシー補足条項（以下「本補足条項」といいます。）は、Kelly SIMONZ Official Fan Club アプリケーション（以下「本アプリケーション」といいます。）のプライバシーポリシーを補足するものであり、本アプリケーション、および本アプリケーションに関連するサービス（以下「本サービス」といいます。）の管理者（以下「管理者」といいます。）に適用されます。本補足条項では、管理者から収集されるデータの種類、収集の目的、およびそのデータが本サービス提供のためにどのように使用または共有されるかについて記載しています。管理者として本アプリケーションを使用することで、本補足条項に記載されている内容に同意したものとみなされます。

本補足条項は、本アプリケーションの "Privacy Policy (URL)" or Hyperlink と併せてお読みください。矛盾が生じる場合、管理者には本補足条項に記載されている内容が優先されます。

第2条（当社が収集する追加情報とその利用方法）

管理者に関して、本アプリケーションの "Privacy Policy (URL)" or Hyperlink に記載されている情報に加え、以下の情報を収集し保存します。

  • Facebook トークン: Facebook アカウントを本アプリケーションにリンクした際に、Facebook トークンが保存されます。このトークンにより Facebook の投稿を本アプリケーションのユーザーと共有できるようになります。アプリケーション内の管理パネルから Facebook アカウントへのアクセスを取り消すことができます。トークンは安全に保管され、アクセスを取り消すと直ちに削除されます。
  • YouTube データ: YouTube アカウントを本アプリケーションにリンクした際に、YouTube トークンが保存されます。このトークンにより YouTube チャンネルへの動画のアップロードができるようになります。YouTube チャンネルへのアクセスは、アプリケーション内の管理パネル、または Google のセキュリティで設定を変更することで取り消すことができます。また、本アプリケーション内で動画をユーザーに表示するために、アップロード後に作成された動画 ID も保存します。30 日以内に同意が得られない場合、YouTube のプライバシーポリシーに従ってこれらの動画へのリンクが削除されます。YouTube のアップロード機能を使用することで、YouTube とその第三者がクッキー（Cookie）または同様の技術を使用して情報を保存、アクセス、または収集することに同意したものとみなされます。
  • アップロードされたコンテンツ: 本アプリケーションにアップロードされた画像は、Google Firebase が提供する Firebase Cloud Storage に保存されます。すべてのデータは日本で安全に保管され、許可されたユーザーまたは本アプリケーションのみがアクセスできるよう、暗号化と厳格なアクセス制御によって保護されています。コンテンツは、収集された目的に必要な限り保管され、要求に応じて削除されます。
  • イベント情報: 本アプリケーション内でイベントを作成し、それを第三者のチケット販売ウェブサイトにリンクすると、イベントの詳細が収集され、Firestore に保存されます。
  • メールアドレス: 機密性の高い操作 (YouTube アカウントの接続、メールや通知の送信、イベントの投稿など) を実行すると、メールアドレスが保存されます。この情報は、これらの操作を円滑に進め、セキュリティを確保するために使用されます。
  • 通信内容: 本アプリケーションから送信されたすべての通信 (メールおよび通知) の内容を記録します。これらの記録は安全に保管され、法的義務の履行を証明するためや法的紛争の際に使用されます。データは、法的に定められた期間、または上記の目的で必要がなくなる時点まで保管されます。

本アプリケーションのプライバシー保護に関するご質問は、当社までご連絡ください。

第3条（追加の許可リクエスト）

管理者として本サービスにアクセスする場合、ファイルピッカー機能を使用するためにデバイスのストレージへのアクセスをリクエストする場合があります。この機能はギャラリーから Firebase Cloud Storage に画像をアップロードしたり、YouTube チャンネルに動画をアップロードしたりするために使用されます。このアクセスは、選択したファイルに厳密に制限されており、プラットフォームにアップロードされたコンテンツを管理するために必要です。デバイスのストレージにあるその他のデータにはアクセスもせず、収集することもありません。

このアクセスは、ファイルピッカーを使用する際にお客様に明示的に同意いただいた上で実行されます。このアクセスは、デバイスの設定からいつでも管理または取り消すことができます。YouTube にアップロードされた動画は、Googleのプライバシーポリシーと YouTube の利用規約に従って取り扱われます。詳細については、それぞれのポリシーと利用規約をご確認ください。

Firebase Cloud Storage にアップロードされたファイルは、Google のプライバシーポリシーと Google Cloud Platform の利用規約に従って取り扱われます。詳細については、それぞれのポリシーと利用規約をご確認ください。

アップロードなどのデータ転送時にもデータが保護されるよう、HTTPS 暗号化や安全な API の呼び出しなどのセキュリティ対策を講じています。

第4条（データの保管）

当社は日本の個人情報保護法（APPI）に従い、本補足条項に記載されたサービスを提供するため、または法的義務を履行するために必要な期間のみ、お客様の Facebook トークン、YouTube のデータ、アップロードされたコンテンツ、およびその他の関連するデータを保管します。

  • Facebook トークンと YouTube に関連するデータ: これらのトークンと関連データは、お客様のアカウントが本アプリケーションにリンクされている期間、本補足条項に記載されている目的のために厳密に保管されます。
  • アップロードされたコンテンツ: アップロードされた画像、動画、その他のコンテンツはイベントや投稿の表示などに必要な限り保管されます。管理者は、本アプリケーション内からいつでも投稿を非表示にすることができます。また、Contact Informationに問い合わせて削除を依頼することもできます。このリクエストは、当社のデータ削除プロトコルに従って速やかに処理されます。
  • メールアドレスと通信内容: メールアドレスと本アプリケーションから送信された通信内容は、メールと通知の送信をするため、法的義務を履行するため、または法的紛争を解決するために必要な限り保管されます。
  • イベント情報: イベントの詳細は、イベントが有効である限り、あるいは管理者またはリクエストによって削除されるまで保管されます。

第5条（お客様の権利）

管理者は、個人情報保護法 (APPI) に従って、個人データにアクセス、修正、または削除を要求する権利があります。これらの権利を行使する場合や、当社のデータ保管への取り組みについて問い合わせる場合には、Contact Informationにご連絡ください。当社は、データの削除を安全に行い、保管するデータは不正アクセス、漏洩、および紛失から保護するために堅牢なセキュリティ対策を講じています。

第6条（本補足条項の変更）

本補足条項は、当社の個人情報保護に関する取り組みや法令の変更、またはその他の業務上の理由に応じて、随時変更される場合があります。変更はすべて、メールまたはその他の適切な手段でお客様に通知されますので、更新された補足条項をご確認ください。

特に指定がない限り、変更は通知から 30 日後に有効になります。変更の効力発生日以降も本アプリケーションを引き続き使用した場合、更新された補足条項に同意したものとみなされます。更新された内容に同意しない場合は、Contact Informationに問い合わせ、データの削除を要求する権利があります。

当社はお客様のプライバシーを保護することに尽力しており、変更が個人情報保護法 (APPI) およびその他の適用される法令を遵守するよう努めます。
