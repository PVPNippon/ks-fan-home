アプリケーションプライバシーポリシー

第1条（定義）

このアプリケーション・プライバシーポリシー（以下、本プライバシーポリシーといいます）は、株式会社ピーブイピー（東京都港区芝大門2−1−16 ＋SHIFT芝大門 代表取締役 石田 桂一、以下、当社といいます）が提供するアプリケーション「Kelly SIMONZ Official Fan Club」（以下、本アプリケーションといいます）および本アプリケーションに関するサービス（以下、本サービスといいます）が取得する情報の取扱いを定めたものです。

第2条（法令等の遵守）

本アプリケーションおよび本サービスは、情報の取得、利用その他の情報の取り扱いについて、個人情報の保護に関する法律、電気通信事業法、関連法令、ガイドライン及びこのプライバシーポリシーを順守します。

第3条（安全管理措置）

本アプリケーションおよび本サービスにより取り扱われる情報は、正確かつ最新の内容に保つよう努めるとともに、不正なアクセス、改ざん、漏洩、滅失及び毀損から保護するため、必要かつ適切な以下の安全管理措置を講じます。

講ずる措置1）基本方針の策定
保有個人データの適正な取扱いの確保のため、「関係法令・ガイドライン等の遵守」、「質問及び苦情処理の窓口」等について「個人情報保護方針」を策定しています。

講ずる措置2）保有個人データの取扱いに係る規律の整備
保有個人データは取得、利用、保存、提供、削除・廃棄等の段階ごとに、取扱方法、責任者・担当者及びその任務等について個人情報保護規程を策定しています。

講ずる措置3）組織的安全管理措置
①保有個人データの取扱いに関する責任者を設置するとともに、保有個人データを取り扱う従業者及び当該従業者が取り扱う保有個人データの範囲を明確化し、法や取扱規程に違反している事実又は兆候を把握した場合の責任者への報告連絡体制を整備しています。
②保有個人データの取扱状況について、定期的に自己点検を実施するとともに、他部署や外部の者による監査を実施しています。

講ずる措置4）人的安全管理措置
①保有個人データの取扱いに関する留意事項について、従業者に定期的な研修を実施しています。
②保有個人データを含む秘密保持に関する誓約書の提出を全従業者から受けています。

講ずる措置5）物理的安全管理措置
①保有個人データを取り扱う区域において、従業者の入退室管理及び持ち込む機器等の制限を行うとともに、権限を有しない者による保有個人データの閲覧を防止する措置を講じています。
②保有個人データを取り扱う機器、電子媒体及び書類等の盗難又は紛失等を防止するための措置を講じるとともに、事業所内の移動を含め、当該機器、電子媒体等を持ち運ぶ場合、容易に保有個人データが判明しないよう措置を講じています。

講ずる措置6）技術的安全管理措置
①アクセス制御を実施して、担当者及び取り扱う個人情報データベース等の範囲を限定しています。
②保有個人データを取り扱う情報システムを外部からの不正アクセス又は不正ソフトウェアから保護する仕組みを導入しています。

第 4 条（データ管理）

第 4-1 条（取得される情報の項目、利用目的、取得方法）

本アプリケーションおよび本サービスのご利用に際して、以下の情報を以下の利用目的のためにアプリケーション経由で自動的に取得いたします。

第4-1-1条（当社が取扱うご利用者の個人情報）

会員登録とログインのプロセスを円滑に行うため、また、メールやオンライン決済などの通信および取引における識別子として、ご利用者のメールアドレスを取得します。さらに、Google アカウントを使用して接続する場合、当社はご利用者の Google アカウントのプロフィール写真を使用し、より快適なユーザー エクスペリエンスを提供します。

第 4-1-2 条（第三者から提供される情報）

本アプリケーションは、サービスの品質を向上させ、データの安全性確保のために、ユーザー情報を安全に処理します。ユーザーデータ（メールアドレスやデバイストークン等）は、信頼できる第三者のチャネルを介して安全に送信され、取引とプッシュ通知を円滑に行うために当社のバックエンドデータベースに保存されます。

当社は、サービスの適切な運用と効率的な提供のために、特定の状況下で第三者と個人情報の共有を行います。

Suzuri: このアプリケーションおよびこのサービスでは、Kelly SIMONZ のショップページから Suzuri 製品が表示されます。アプリケーションに表示された製品を操作すると、Suzuri の製品購入ページにリダイレクトされます。Suzuri の Web サイトは、このサービスおよびこのアプリケーションとは別のサービスであり、当社が管理するものではありません。日本国外からの接続の場合、Suzuri は台湾からのユーザーに対しては Bibian のサービスを使用し、それ以外の地域からのユーザーに対しては WorldShopping のサービスを使用して、それぞれチェックアウトと注文を管理することにご注意ください。

これらのサービスは Suzuri とは異なります。お客様の権利を理解するために、お客様が操作するさまざまなサービスのプライバシー ポリシーと利用規約を注意深くお読みになることをお勧めします。当社は、Suzuri、WorldShopping、Bibian のいずれかで行われた取引、支払い、注文に関連する情報を収集、保存、またはアクセスすることはありません。

Sendgrid: ご利用者のメール アドレスは、当社のメール送信要件を満たすために Sendgrid サービスと共有されます。Sendgrid は、本サービスからのメール受信に同意したご利用者とのメール送受信の管理を支援する信頼できる第三者のサービスプロバイダです。当社のメール購読を解除したい場合は、メールに含まれる配信停止リンクをクリックするか、本アプリケーション内の「プロフィール」＞「メール通知設定」で、いつでも解除することが可能です。

Facebook: このアプリケーションおよびこのサービスは Facebook と統合され、Facebook の投稿などのコンテンツを表示しています。

当社は、本アプリケーション内で Facebook の投稿を表示するために必要な Facebook のプラットフォーム データのみを処理します。これには、ご利用者が本アプリケーションの使用中に表示または操作するすべてのコンテンツが含まれます。ご利用者の Facebook アカウントからその他の個人情報を取得または保存することはありません。

当社は Facebook のプラットフォーム データを上記の目的にのみ使用し、第三者と共有することはありません。

当社は Facebook のプラットフォーム利用規約およびその他の適用されるすべての規約とポリシーを厳守します。本アプリケーションの Facebook プラットフォーム データの使用は、上記の目的にのみ使用されます。Facebookの利用規約とポリシーの詳細については、こちら をご確認ください。

Google Maps API: このアプリケーションとこのサービスは、地図ベースのサービスを提供するために Google Maps API を使用します。Google マップ 使用時に、ユーザーの位置情報データを取得または保存することはありません。本アプリケーション内での Google マップの利用は、管理者が作成したイベントの地図へのリンクを提供するものに限定されています。本アプリケーションおよび本サービス内で Google マップが提供する地図を操作する場合、データ処理は Google によって直接行われます。Google マップは独自のセキュリティ対策を講じて運営されています。Google マップによって処理されるデータは、Google のプライバシー ポリシーとセキュリティ プロトコルの対象となります。Google マップの機能を使用する場合、Google マップの プライバシーポリシー および Google マップ/Google Earth 追加利用規約に同意したものとみなされます。本サービス内で地図を操作すると、Google マップがご利用者のモバイルデバイスのストレージにアクセスし、クッキー（Cookie） や同様の技術にアクセスまたは認識する場合がありますのでご留意ください。

Crashlytics: このアプリケーションとこのサービスは、Google Firebaseが提供するサービスであるCrashlyticsを使用して、安定性とパフォーマンスを監視します。Crashlytics は、アプリケーションのクラッシュやその他の技術的エラーの特定や解決を行い、ユーザーエクスペリエンスを向上させます。Crashlytics は、個人識別用情報 (PII) や機密データを取得することはありませんが、クラッシュやその他のエラーが発生した際には、デバイスとアプリケーションの使用状況に関する特定のデフォルトデータを取得します。取得された情報は、技術的な問題の診断と修正等の、アプリケーションのパフォーマンスと安定性を向上させる目的のみに使用されます。

Crashlytics が取得した情報の取り扱いについての詳細は、Google Firebase のプライバシーとセキュリティのページをご確認ください。

Firebase Cloud Messaging (FCM): このアプリケーションとこのサービスは、Google Firebaseが提供するサービスであるFirebase Cloud Messaging（FCM）を使用して、サービスに登録した場合にデバイスに通知とメッセージを送信します。FCM は個人識別用情報 (PII) や機密データを取得することはありませんが、デバイスに通知やメッセージを送信するために特定のデフォルト データを取得します。 (下記 第4-2-1条 と 第4-2-2条 をご覧ください。) 取得された情報は、通知を送信し、対象のデバイスに確実に届ける目的のみに使用されます。

FCM が取得した情報の取り扱いについての詳細は、Google Firebase のプライバシーとセキュリティのページをご確認ください。

Firebase Remote Config: このアプリケーションとこのサービスは、Google Firebase が提供するサービスである Firebase Remote Config を使用して、アプリの機能を動的に管理およびカスタマイズします。Firebase Remote Configを使用すると、このアプリケーションの新しいバージョンをダウンロードしなくても、アプリの設定と機能を更新できます。Firebase Remote Config は、個人識別用情報 (PII) や機密データを取得することはありませんが、アプリケーションのコンフィギュレーションを提供するために、デバイスに関する特定のデフォルトデータを取得します。収集された情報は、このアプリケーションとこのサービスの機能およびユーザーエクスペリエンスを最適化する目的でのみ使用されます。

Firebase Remote Config が取得した情報の取り扱いについての詳細は、Google Firebase のプライバシーとセキュリティのページをご確認ください。

Firebase Cloud Firestore: 本アプリケーションおよび本サービスは、Google Firebase が提供するサービスである Firebase Cloud Firestore を使用して、ユーザー プロファイルなどの構造化データや、その他のアプリ関連データを保存および管理します。すべてのデータは日本国内に保存されますが、ユーザーは本アプリケーションおよび本サービスに世界中からアクセスできます。Firebase Cloud Firestore は、本サービスの運用に必要な範囲を超えて個人識別用情報 (PII) を取得することはありません。すべてのデータは安全に保存され、承認されたユーザーまたは本サービスのみがアクセスできます。

Firebase Cloud Firestore が保存する情報の取り扱いについての詳細は、Google Firebase のプライバシーとセキュリティのページをご確認ください。

Firebase Analytics: このアプリケーションとこのサービスは、Google が提供するサービスである Firebase Analytics を使用して、このアプリケーション内でのユーザーの操作を収集および分析します。Firebase Analytics によって収集されるデフォルトのデータは、ユーザーがこのアプリケーションとこのサービスとどのようにやり取りしているかを把握するために使用され、ユーザー エクスペリエンスの向上、アプリのパフォーマンスの最適化、潜在的な問題の特定が可能になります。

Firebase Analytics が取得したデータは Google と共有され、Google のプライバシー ポリシーに従って使用される場合があります。当社は、パーソナライズド広告の目的でこのデータを使用することはありません。Firebase Analytics が取得したデータの取り扱いについての詳細は、Google のプライバシー ポリシーをご確認ください。

当社は商業的に妥当な手段を用いてご利用者の個人データの保護に努めていますが、インターネット経由の送信方法や電子的な保存方法については 100% の安全性を保証することはできません。
